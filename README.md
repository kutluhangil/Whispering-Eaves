# 🏛️ WHISPERING EAVES ( Whispering Eaves — 听檐 / 囁く庇 )

> A single-file, full-screen interactive web editorial artwork inspired by building eaves and Verlet text curtain physics, themed around a historical Silk Road cultural journey across China, Japan, Kazakhstan, Uzbekistan, and Türkiye.

## 🌟 Highlights & Features

- 🏛️ **5 Silk Road Destinations:** China (*Xi'an Palace Eaves*), Japan (*Nara Shinto Shrine Roof*), Kazakhstan (*Altay Yurt Shanyrak*), Uzbekistan (*Samarkand Registan Portal*), and Türkiye (*Istanbul Ottoman Mansion*).
- 📜 **Verlet Text Curtain Physics:** Local-script glyphs hanging on spring-connected Verlet physics grids. Touch, swipe, drag, burst, and wave interactively.
- 🏺 **3D WebGL Cultural Artifact Inspector:** Interactive 3D wireframe models of historical artifacts (Tang Tricolor Horse, Torii Guard, Golden Warrior Helmet, Samarkand Turquoise Amphora, Iznik Pitcher).
- 🎙️ **Poetry Recitation & Voiceovers:** Native poetry recitations (Li Bai, Bashō, Abay, Alisher Navoi, Yunus Emre) with SpeechSynthesis audio voiceovers.
- 🗺️ **Interactive Silk Road Route Map:** Vector map tracking trade caravan paths with teleportation nodes.
- 🎹 **Keyboard Instrument Synthesizer:** Play ethnic pentatonic chime melodies using keys `1-9` and `A-K`.
- 📸 **Editorial Poster Exporter:** Generate and download high-resolution PNG posters of any scene.
- 🧘 **Zen Mode:** Press `Z` to hide chrome UI elements for a pure, meditative artwork view.
- ❄️ **Seasonal Weather Engine:** Toggle between Spring Sakura, Autumn Leaves, Winter Snow, and Rainy Night particle systems.
- 🌙 **Night Mode:** Toggle ambient charcoal paper and glowing architectural lantern modes.

## 🚀 Quick Start

No build tools, Node packages, or backend required! Simply open `silk-atlas/index.html` in any web browser or serve it locally:

```bash
# Serve locally with Python
python3 -m http.server 5173 --directory silk-atlas
```

Then visit `http://localhost:5173/` in your browser.
